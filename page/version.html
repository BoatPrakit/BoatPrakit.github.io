Version

- Version 1.5.2
    เพิ่มเติมในระบบการเล่นมายคราฟออนไลน์ รวมถึงการปรับในส่วนของ Zombie และมีการเพิ่มบล็อคอย่าง Hardened Clay

- Version 1.6.4
    เพิ่มและลดในส่วนของฝั่ง Server และ Client สิ่งที่เพิ่มขึ้นมาคือเรื่องของ การกำหนด timeout ของผู้เล่นประมาณ ว่าถ้าไม่ขยับหรือไม่ทำอะไรเลยในเวลาที่ กำหนดนั้นจะถูกเตะของจากเซิฟเวอร์

- Version 1.7.2
    อัพเดทในเรื่องของ World Generation ซึ่งเวลาสร้างโลกขึ้นมาใหม่นั้นมายคราฟ เวอร์ชั่นนี้จะมีการปรับเปลี่ยนระบบนิเวศและ ต้นไม้ต่างๆ เยอะพอสมควร และมีส่วนของ ไอเท็มที่เพิ่มเข้ามาอีก และเวอร์ชั่นนี้มีการ แก้ไขบัคเกือบร้อยตัว

- Version 1.8.9
    มีการเพิ่มการแจ้งเตือนอาณาจักรไปที่เมนู Options รูปแบบกล่องเก็บของจะเปลี่ยน ให้เข้ากับเทศกาลคริสต์มาส และมีการแก้ไข บัค 1 บัคที่ผู้เล่นติดและไม่สามารถออกจาก น้ำไหลได้

- Version 1.10.2
    เวอร์ชั่นนี้ไม่ได้มีอะไรอัพเดทที่แตกต่างกันไป จากเดิมมากนัก แต่เป็นที่นิยม

- Version 1.12
    การปรับปรุงมายคราฟ 1.12 นี้จะเพิ่มระบบ งานคราฟชิ้นใหม่, ระบบฟังก์ชั่นใหม่, คำแนะนำ,สำหรับผู้เริ่มต้น , glazed terracotta , คอนกรีต, เตียงสี, ภาพลวงตาและนกแก้ว
